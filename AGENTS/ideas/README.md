# Ideas Layer

This directory is the backlog and product-memory layer.

Use it for:

- ideas not yet ready for implementation
- options that need more discovery
- ideas that were consciously rejected

Do not use `ideas/` for step-by-step implementation notes. Once work becomes active, create a task
file under `AGENTS/tasks/`.

Taxonomy:

- `candidate/`
  - reasonable next ideas, but not active tasks yet
- `needs-discovery/`
  - promising directions that still need research or sizing
- `rejected/`
  - ideas the project deliberately does not want right now
